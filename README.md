# Fethiye Real Estate Market Analyzer
An end-to-end data science project designed to analyze the real estate market in Fethiye (scalable to other cities) using data analysis techniques and machine learning algorithms. 

## Project Goal
The primary objective is to build an automated system that generates statistical analytics, detects undervalued investment opportunities, and performs realistic real estate valuations using market data.

## Tech Stack
This project uses the following technologies for the ETL (Extract, Transform, Load) pipeline and data analysis:
- **Data Collection (Scraping)**: Selenium
- **Data Processing**: NumPy, Pandas, RegEx
- **Database**: PostgreSQL
- **Visualization & Analysis**: Matplotlib, Seaborn
- **Machine Learning**: Scikit-learn(Linear Regression, Random Forest...)
